# MVP - Minimum Viable Product

- Hackreactor project where we had to make a minimum viable product on any topic of choice in 48 hours.

**Built With**

-React.js
-React-Bootstrap
-Node.js
-Express
-Webpack

-npm
-npm install npm@latest -g
